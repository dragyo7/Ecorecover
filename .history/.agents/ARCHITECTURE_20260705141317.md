# EcoRecover Architecture
Version: 1.0

## Vision

EcoRecover is a scalable, AI-powered e-waste recycling platform that provides intelligent valuation, recycler discovery, pickup scheduling, user rewards, and analytics through a modern Android application backed by FastAPI.

---

# Architecture

Android
↓
Repository Layer
↓
FastAPI
↓
Service Layer
↓
Repository Layer
↓
PostgreSQL / Supabase

Every layer communicates only with its adjacent layer.
