# Project Identity

Project Name

EcoRecover

Platform

Android

Backend

FastAPI

Purpose

An on-demand e-waste pickup platform inspired by Blinkit, Instamart and Uber.

Users recycle electronics instead of buying products.

The experience prioritizes convenience, transparency and trust.

------------------------------------------------------------------------------

# Product Vision

The application should make selling old electronics as easy as ordering groceries.

The application should provide:

• Instant search

• Live valuation

• Transparent pricing

• Pickup scheduling

• Live tracking

• Rewards

• Sustainability insights

# Things AI Must Never Forget

This is not an e-commerce application.

This is not a marketplace.

This is not a dashboard.

Users SELL electronics.

Users do not buy products.

The valuation engine is the heart of the application.

The pickup experience is the primary user journey.

Transparency builds trust.

Backend owns business logic.

Android presents backend data.

Extend.

Do not replace.

Reuse.

Do not duplicate.

Maintain one source of truth.
